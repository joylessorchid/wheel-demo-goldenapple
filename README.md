# 🎡 Колесо Удачи — Golden Apple

https://wheel-demo-goldenapple.ibuypower-xe10.workers.dev/
Промо-лендинг с интерактивным колесом фортуны, стилизованный под фирменный язык **Golden Apple**: минималистичная типографика на Inter, мягкая пастельная палитра секторов, чёрно-белый интерфейс.

Лёгкий статический сайт без бэкенда и сборки — три файла, ~29 КБ суммарно, открывается двойным кликом по `index.html` либо через любой статический сервер.

---

## ✨ Возможности

- **Колесо на `<canvas>`** — 10 секторов, плавная анимация остановки (cubic ease-out), стрелка-указатель на 12 часов.
- **Взвешенный розыгрыш** — 5 призов с весами 35 / 25 / 20 / 15 / 5 %; результат определяется *до* анимации, колесо останавливается ровно на нужном секторе.
- **Модалка с промокодом** — иконка, название приза, описание, копируемый код.
- **Реферальная механика** — после ввода e-mail генерируется уникальная ссылка вида `?ref=ABC123`; шеринг в Telegram / VK / WhatsApp даёт +1 попытку.
- **Соц-доказательство** — счётчики «участников» и «призов осталось» обновляются в реальном времени.
- **Таймер обратного отсчёта** до конца акции в верхнем баннере.
- **Адаптив** — 3 брейкпоинта (десктоп / 900 px / 480 px), на мобильных колесо сжимается до 340 px.

## 🚀 Запуск

Самый простой вариант — открыть `index.html` напрямую в браузере.

Для корректной работы `navigator.clipboard` (копирование реферальной ссылки) и production-окружения лучше отдавать через локальный сервер:

```bash
# Python
python3 -m http.server 8000

# Node
npx serve .
```

Затем `http://localhost:8000`.

## ⚙️ Настройка призов

Призы и их вероятности задаются в `script.js`:

```js
const UNIQUE = [
    { ...SEGMENTS[0], weight: 35 },  // Скидка 5%
    { ...SEGMENTS[2], weight: 25 },  // Скидка 10%
    { ...SEGMENTS[4], weight: 20 },  // Бесплатная доставка
    { ...SEGMENTS[1], weight: 15 },  // Скидка 30%
    { ...SEGMENTS[3], weight:  5 },  // Бесплатный товар
];
```

`SEGMENTS` — это 10 видимых секторов колеса (каждый приз продублирован дважды для визуального баланса). Чтобы изменить призы:

1. Отредактируй `SEGMENTS` (label / code / desc / color).
2. Подкорректируй веса в `UNIQUE`.
3. Обнови `segMapping` в функции `spin()`, если меняешь порядок призов.
4. Синхронизируй список и проценты в `<div class="prizes-list">` в `index.html`.

## 📝 Особенности реализации

- Колесо отрисовывается полностью на `<canvas>`, без SVG-шаблона — это упрощает анимацию и масштабирование.
- Призовой исход выбирается **до** запуска анимации (`pickPrize()` возвращает индекс), и угол поворота вычисляется так, чтобы выбранный сектор оказался под указателем — никаких «сначала анимация, потом считаем результат».
- Easing `1 − (1 − p)⁴` даёт характерное физическое замедление колеса в конце.
- Иконка приза в модалке выбирается по `code`: `🎁` — товар, `🚚` — доставка, `💎` — скидка.
